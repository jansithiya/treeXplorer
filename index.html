Visual Interfaces

DepthCollapse

Provide collapsed view of phylogenetic tree beyond chosen depth. Triangle appended to node indicates the collapsed node and the area of triangle indicates number of leaf nodes hidden under the node

Explore!

Hybrid Tree

Visualize the phylogenetic tree in hybrid layout which is the combination of node-link and treemap. Provides collapsed view of phylogenetic beyond chosen depth and all the children under are shown in treemap layout

Explore!

Degree of Interest Tree

Ideal for exploring large phylogenetic tree based on interestingness score. Only interesting nodes corresponding to user selected focus node gets displayed on screen while maintaing overall context

Explore!

TreeMap

Treemap is space-constrained visualization suitable for hierarchical data. Also an alternative and novel way to visualize very large phylogenetic tree.
